1.4 Retry 重试工具 — backend/app/agents/retry.py
----------------------------------------------------------------------

  llm_call_with_retry(messages, temperature, max_tokens, max_retries, timeout)
    │
    ├── llm = get_llm(temperature, max_tokens)
    │
    ├── for attempt in range(max_retries + 1):
    │    ├── response = await asyncio.wait_for(
    │    │       llm.ainvoke(messages), timeout=timeout)
    │    ├── 成功 → return response.content
    │    └── 失败 → await asyncio.sleep(min(2^attempt, 8))
    │
    └── 全部失败 → raise Exception

  parse_json_response(content)
    └── 去除 ```json / ``` 标记 → json.loads

3.4 匹配算法详解
----------------------------------------------------------------------

  MatchScorer.compute_scores(profile, job_reqs, job_info)

  输入:
    profile    用户画像 (7维雷达数据)
    job_reqs   岗位要求 (7维期望分数, 由 job_profiler 提取)
    job_info   岗位元信息

  计算逻辑:
    ┌────────────────────────────────────────────────────┐
    │  对每个维度 i (共7个):                               │
    │    user_score  = profile.radar_data[i]              │
    │    expected    = job_reqs[dim].expected_score       │
    │    gap         = expected - user_score              │
    │                                                     │
    │  如果 gap <= 0:                                      │
    │    dim_score = 100 - |gap| * 0.3  (超出越多越接近100) │
    │  如果 gap > 0:                                       │
    │    dim_score = max(0, 100 - gap * 1.2)  (差距扣分)   │
    │                                                     │
    │  total_score = Σ(dim_score * weight[i])             │
    │  权重: [0.18, 0.13, 0.15, 0.16, 0.10, 0.10, 0.18]  │
    └────────────────────────────────────────────────────┘

  输出:
    {
      "job_id": str,
      "job_title": str,
      "company": str,
      "total_score": float,         # 加权总分
      "scores": {                   # 7维详情
        "专业技能": {"score": 75, "expected": 80, "gap": "需提升5分"},
        ...
      }
    }
